# Milestone v1.0: Quality MVP

**Status:** ✅ SHIPPED 2026-03-11
**Phases:** 1-4
**Total Plans:** 12

## Overview

This milestone addressed quality, not growth. The work moved in a deliberate order: first audit to find everything broken, then fix actual bugs, then clean up structural debt, then unify the visual and mobile experience. Each phase delivered a verifiable improvement to the app's reliability and polish.

## Phases

**Depends on**: Nothing (first phase)
**Requirements**: AUDT-01
**Plans**: 3 plans

Plans:
- [x] 01-01: Audit learning pages: index.html, topic.html, sentences.html, conjugation.html, fill-blank.html
- [x] 01-02: Audit standalone pages: vocab.html, voices.html

## Milestone Summary

**Key Decisions:**

- Audit before fixing: Don't assume scope — find all issues systematically before fixing anything (✓ Validated — audit revealed 8+ issues not previously documented)
- Keep IIFE pattern: Migration to ESM not worth cost for project size (✓ Validated — refactoring kept clean with IIFE; no friction encountered)
- Consolidate TSV loaders into SharedUtils: Reduces duplication, single place to fix future TSV changes (✓ Validated — SharedUtils.loadTSV now generic loader for fill-blank and conjugation)
- scheduleMusic in loop() is correct iOS keepalive, not a per-frame bug (✓ Validated — CONCERNS.md corrected)
- :root CSS variables declared inline in jungle-run.html (not linking styles.css) to preserve full-screen canvas layout (✓ Validated — pattern established for canvas game pages)
- min-height:44px added to specific button classes individually, never to base .btn (✓ Validated — quiz-nav-row compact buttons preserved)

**Issues Resolved:**

- Quiz back button coin-refund exploit (BUG-01)
- iOS/Safari voice loading silent failure (BUG-02)
- Game lives silent redirect to / on direct URL access (BUG-03)
- Missing favicon on all 11 pages (BUG-04)
- localStorage quota exceeded with no user feedback (BUG-05)
- voices.html crash on browsers without speechSynthesis API (BUG-06)
- Inaccurate CONCERNS.md scheduleMusic entry (corrected)

**Technical Debt Incurred:**

- In-game scores (coins collected in mini-games) not connected to persistent CoinTracker economy — deferred to v2
- User state keyed by text strings (practice list, enabled sentences) — survives vocab edits poorly — deferred to v2
- TSV caching strategy: still using cache: "no-store" — deferred to v2
