# Milestone v1.8: Content & Settings

**Status:** ✅ SHIPPED 2026-04-24
**Phases:** 15-16
**Total Plans:** 2

## Overview

Added 6 new Spanish verbs to the conjugation practice pool and replaced the Build Sentences per-sentence toggle list with per-category checkboxes — making it practical to enable/disable a whole vocabulary unit with a single interaction.

## Phases

### Phase 15: Verb Data Entry

**Goal:** verbs.tsv contains 6 new verbs with full conjugation tables so users can practice them in the Verbs mode immediately.
**Depends on:** Phase 14
**Requirements:** DATA-02
**Plans:** 1 plan

Plans:
- [x] 15-01: Append 6 new verbs to verbs.tsv (saber, hacer, beber, vivir, entender, comer)

**Details:** Single TSV data-entry plan. No JS/HTML changes needed — conjugation.js already loads all rows from verbs.tsv dynamically. CRLF line endings and UTF-8 NFC accents preserved.

---

### Phase 16: Build Sentences Category Filter

**Goal:** The Build Sentences settings panel lets users filter by category rather than toggling individual sentences.
**Depends on:** Phase 15
**Requirements:** SENT-01
**Plans:** 1 plan

Plans:
- [x] 16-01: Refactor sentences.js and sentences.html settings panel to use per-category checkboxes

**Details:** Added `getCategoryFilter`/`saveCategoryFilter` functions with localStorage persistence under `sentenceCategories` key. `openSentenceManager` now renders one checkbox per category (e.g. "Saludar (10)") instead of one per sentence. `init()` filters active sentences by `filterMap[s.category]`. Modal title updated to "Filter by Category".

---

## Milestone Summary

**Key Decisions:**
- No migration from old `enabledSentences` key — new `sentenceCategories` key is fully independent; old data is silently unused
- Category names rendered via `label.textContent` (XSS-safe); `saveCategoryFilter` wraps `setItem` in try/catch with visible error message
- New categories added to TSV default to `true` (enabled) without requiring user re-configuration

**Issues Resolved:**
- verbs.tsv was missing saber, hacer, beber, vivir, entender, comer — now all 6 are available in Conjugation mode
- Build Sentences settings panel was impractical for large vocabulary sets (required toggling 50+ individual sentences)

**Technical Debt Incurred:**
- `getEnabledSentences` / `saveEnabledSentences` functions remain in sentences.js as dead code — harmless, intentionally kept for backwards compat
- Gear button tooltip still reads "Select Sentences" while modal header reads "Filter by Category" — cosmetic only
