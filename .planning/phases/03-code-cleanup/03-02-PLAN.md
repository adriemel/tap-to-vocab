The file must contain exactly the logic currently in index.html's two inline script blocks, wrapped in a single DOMContentLoaded listener:

```javascript
(function () {
  document.addEventListener("DOMContentLoaded", function () {
    // Practice count badge
    try {
      var list = JSON.parse(localStorage.getItem("practiceList") || "[]");
      var practiceBtn = document.getElementById("practice-btn");
      if (practiceBtn) {
        practiceBtn.textContent = list.length > 0
          ? "\u2B50 Practice (" + list.length + ")"
          : "\u2B50 Practice";
      }
    } catch (e) {}

    // Games button: spend 10 coins to play
    document.getElementById("btn-games").addEventListener("click", function () {
      if (CoinTracker.spendCoins(10)) {
        sessionStorage.setItem("game_lives", "3");
        location.assign("/games.html");
      } else {
        var msg = document.getElementById("coins-msg");
        msg.textContent = "Need 10 coins to play \u2014 keep answering correctly!";
        msg.style.display = "block";
        setTimeout(function () { msg.style.display = "none"; }, 2500);
      }
    });

    // Reset coins button
    document.getElementById("btn-reset-coins").addEventListener("click", function () {
      if (confirm("Are you sure you want to reset your coins to 0?")) {
        CoinTracker.resetCoins();
      }
    });
  });
})();
```

Notes:
- Merge both inline script blocks into one DOMContentLoaded listener — the reset-coins listener was outside DOMContentLoaded in the original, but the button exists in the HTML before the script tag runs, so this is safe and cleaner.
- Use `var` to match the coins.js / IIFE style already established.
- Unicode escapes for ⭐ (\u2B50) and — (\u2014) match the original exactly.
- CoinTracker is loaded by coins.js before home.js (guaranteed by script load order in index.html).
