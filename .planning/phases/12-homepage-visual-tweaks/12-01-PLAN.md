# Phase 12: Homepage & Visual Tweaks

## Goal

Consolidate Tiempo and Idiomas into a visual sub-group under the Palabras button, and lighten the global background color from near-black to a slightly warmer dark navy.

## Context

The homepage grid currently has 12 standalone category buttons in `.grid-two-col`. Tiempo and Idiomas are semantically related to Palabras (general vocabulary words / language topics). Grouping them saves two rows of grid space and signals their relationship visually.

The current `--bg: #0b1020` is very close to pure black, making the UI feel heavy. Bumping it to `#152238` adds visible depth without breaking the dark theme.

READ FIRST: tap-to-vocab/index.html (full file) and tap-to-vocab/assets/css/styles.css (existing .grid-two-col and .btn rules).

In index.html:
- Remove the two standalone anchor tags for Tiempo and Idiomas from the flat .grid-two-col:
    `<a class="btn" href="/topic.html?cat=Tiempo">⏰ Tiempo</a>`
    `<a class="btn" href="/topic.html?cat=Idiomas">🗣️ Idiomas</a>`
- Replace the standalone Palabras anchor with a .palabras-group wrapper that spans both columns:

```html
<div class="palabras-group">
  <a class="btn btn-palabras-main" href="/topic.html?cat=Palabras">🧩 Palabras</a>
  <div class="palabras-sub">
    <a class="btn btn-palabras-sub" href="/topic.html?cat=Tiempo">⏰ Tiempo</a>
    <a class="btn btn-palabras-sub" href="/topic.html?cat=Idiomas">🗣️ Idiomas</a>
  </div>
</div>
```

In styles.css, add after the existing .grid-two-col block:

```css
/* Palabras group — main button + sub-buttons row */
.palabras-group {
  grid-column: 1 / -1;
  display: flex;
  flex-direction: column;
  gap: 6px;
}

.btn-palabras-main {
  width: 100%;
  text-align: center;
  justify-content: center;
}

.palabras-sub {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 6px;
}

.btn-palabras-sub {
  width: 100%;
  text-align: center;
  justify-content: center;
  font-size: 0.88rem;
  padding-top: 7px;
  padding-bottom: 7px;
  opacity: 0.82;
  border-style: dashed;
}
```

The dashed border and reduced opacity visually subordinate the sub-buttons to the main Palabras button without introducing any new color. No JS changes needed.
